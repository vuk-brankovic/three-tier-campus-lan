# Three_Tier_Campus_LAN

Cisco Packet Tracer complete lab and design following a Three-Tier Campus LAN topology.

## Topology

![Campus Topology](screenshots/topology.png)

## Overview

This post-CCNA project was based on my University of Belgrade campus, adapted to lab conditions and CCNA topics.

## Technologies

Management (SSH, VTY Lines) | EtherChannels (L2, L3) | VLANs | VoIP | IP Addressing | Subnetting | HSRP | STP (Rapid PVST+) | STP and HSRP Synchronization | Static and Dynamic Routing (OSPF, Floating Static Routes) | VLAN SVIs (Multilayer Switching) | Network Services (DHCP, DNS, NTP, SNMP, Syslog, SSH, NAT) | CDP | LLDP | ACLs | Security (Port Security, DHCP Snooping, DAI) | Wireless (Lightweight APs, WLC GUI, WLANs)

## Chamber of Secrets

There is a website containing a hidden message hosted on a server inside the Chamber of Secrets. Your task is to find it, starting from any PC in the University except the Admin's PC in his office. Just access the Command Prompt from any PC and start exploring! Follow the leads until you successfully reach the final message. Hint: Try visiting university website: http://university

The solution can be found in the text file Chamber of Secrets.

You don't have to look at any other configuration text files, but if you feel the need, you can. There are no hints in them.

## Configuration Files

There are three configuration files, two containing necessary addressing data and a third containing instructions from zero to setting up this whole lab completely on your own. You can follow it and try it for practice.
